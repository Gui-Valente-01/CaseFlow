# Cerebro do CaseFlow

Este projeto usa um vault do Obsidian como fonte oficial de contexto do produto.

## Caminhos

- Site oficial: `C:\Users\Win 11\Desktop\Sistema de advogado\sistema-advogados`
- Cerebro do projeto: `C:\Users\Win 11\Desktop\Sistema de advogado\CaseFlow\caseflow-brain`

## Como usar

Antes de alterar codigo, consulte o cerebro para entender produto, regras,
permissoes, banco de dados e arquitetura. O `AGENTS.md` deste repositorio lista
as notas principais que devem ser lidas.

## Quando atualizar o cerebro

- Decisao nova de produto, regra de negocio ou arquitetura.
- Mudanca importante no modelo do banco.
- Erro relevante encontrado e resolvido.
- Alteracao de roadmap ou prioridade do MVP.

## Regra pratica

O codigo vive neste repositorio. O contexto vivo do projeto vive no Obsidian.
Quando uma mudanca de codigo alterar uma decisao do produto, atualize os dois.
